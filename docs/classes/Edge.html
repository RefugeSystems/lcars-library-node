contribution

String

Defined in library\elements\edge.js:53

When determining if an element is "in distress" or not, the contribution of all supporting elements is considered during an update.

This must result in a number to be considered valid and is passed the status of the "source" and "target" as 0: Functional, 1: Warning, 2: Exception, or 3: Deactivated .
