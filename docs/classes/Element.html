active

Boolean

Defined in library\elements\element.js:97

Indicates if the element should be considered active.

This does NOT affect it's availability for querying or graph linking, this is simply an extension of the status.

An example would be taking a server offline for maintenance.

See for making an element unavailable to searches and linking.
